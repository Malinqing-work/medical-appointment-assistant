# 合疗报销精简指南

## 一、必备材料

1. **通用材料**：合疗证（或电子合疗凭证）原件及复印件、本人身份证原件及复印件（代办需额外提供代办人身份证）、医院开具的疾病诊断证明书（加盖公章）。

2. **异地就医患者**：异地就医备案表（或线上备案成功截图打印件）、住院或门诊的费用发票原件、费用明细清单。

3. **特殊情形**：

- 住院患者：住院病历复印件（包括入院记录、出院小结等）、出院证明。

- 意外伤害患者：意外受伤经过说明（需村或社区盖章证明）、相关医疗证明材料。

## 二、报销地点

1. **本地就医**：户籍所在地的乡镇（街道）合疗经办窗口或当地政务服务中心合疗专区。

2. **异地就医**：若已办理异地直接结算，在就诊医院结算窗口即可报销；未直接结算的，回户籍所在地的合疗经办机构报销。

## 三、报销流程

### （一）门诊报销

1. 就诊时出示合疗证（或电子凭证），支付门诊费用并索取发票和费用明细。

2. 携带相关材料到户籍所在地合疗经办窗口提交报销申请。

3. 工作人员审核材料，符合报销条件的，按规定比例核算报销金额，通常在一定工作日内将报销费用发放至本人银行账户。

### （二）住院报销

1. 办理住院手续时，向医院出示合疗证、身份证及异地备案材料（异地就医者），进行合疗登记。

2. 出院时，若符合直接结算条件，在医院结算窗口直接扣除合疗报销部分，支付个人自付金额即可。

3. 未实现直接结算的，出院后收集齐住院发票、费用明细、病历复印件、出院证明等材料，回户籍所在地合疗经办窗口提交报销申请，审核通过后领取报销费用。
